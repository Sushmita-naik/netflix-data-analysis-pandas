Netflix Data Analysis using Pandas
Project Overview:
This project analyzes the Netflix Movies and TV Shows dataset using Python and Pandas.
The goal is to explore the dataset, filter information, and perform basic data cleaning.
Dataset:
The dataset contains information about Netflix titles including:
1)Title
2)Type (Movie / TV Show)
3)Country
4)Release Year
5)Cast
6)Director
7)Description
8)Source: Netflix dataset (Kaggle)
Technologies Used:
Python
Pandas
CSV Dataset
Analysis Performed:
The following operations were performed on the dataset:
1)Loaded dataset using Pandas
2)Explored dataset structure
3)Displayed first 10 rows
4)Counted number of Movies and TV Shows
5)Filtered movies released after 2021
6)Filtered movies produced in the United States
7)Removed unnecessary columns
8)Saved cleaned dataset
