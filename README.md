Netflix Data Analysis using Pandas
Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python and Pandas.
The goal is to explore the dataset, filter information, and perform basic data cleaning.

Dataset

The dataset contains information about Netflix titles including:

Title

Type (Movie / TV Show)

Country

Release Year

Cast

Director

Description

Source: Netflix dataset (Kaggle)

Technologies Used

Python

Pandas

CSV Dataset

Analysis Performed

The following operations were performed on the dataset:

Loaded dataset using Pandas

Explored dataset structure

Displayed first 10 rows

Counted number of Movies and TV Shows

Filtered movies released after 2021

Filtered movies produced in the United States

Removed unnecessary columns

Saved cleaned dataset
